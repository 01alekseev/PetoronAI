# PetoronAI

**PetoronAI** is a deterministic cognitive AI framework built entirely on top of **PetoronHash2 (PH2)** and **zkPetoron-Core**.

The project is designed as a local-first, privacy-first cognitive architecture focused on reasoning, planning, learning, memory, simulation, and autonomous decision making.

PetoronAI provides its own cognitive foundation that developers can use to build autonomous agents, intelligent systems, decision engines, research platforms, and future AI operating environments.
